# Sambhav Khandelwal Resume Website

Static portfolio website package with:
- `index.html`
- `styles.css`
- `script.js`
- `assets/images/*`
- `assets/pdfs/*`

## Deploy on GitHub Pages
1. Upload the full folder contents to your repository root.
2. Commit and push.
3. In GitHub repo settings, enable **Pages**.
4. Choose the main branch and root folder.
5. Save.

Because all paths are relative, the site will work on GitHub Pages without changing the links.
